# User Stories

## 一覧

### US-001: タスクを追加する

**As a** 個人ユーザー
**I want** テキストを入力してタスクを追加したい
**So that** やるべきことを忘れずに記録できる

**Acceptance Criteria:**

- [ ] AC1: テキスト入力欄にタスク名を入力し、Enterキーまたはボタンで追加できる
- [ ] AC2: 追加したタスクが一覧に即座に表示される
- [ ] AC3: 空のテキストではタスクを追加できない

**優先度:** Must

---

### US-002: タスク一覧を見る

**As a** 個人ユーザー
**I want** 未完了タスクの一覧を見たい
**So that** 今やるべきことを把握できる

**Acceptance Criteria:**

- [ ] AC1: アプリを開くと未完了タスクの一覧が表示される
- [ ] AC2: タスクは追加した順に表示される

**優先度:** Must

---

### US-003: タスクを完了する

**As a** 個人ユーザー
**I want** タスクを完了済みにしたい
**So that** 終わったことが明確になる

**Acceptance Criteria:**

- [ ] AC1: タスクのチェックボックスをクリックして完了にできる
- [ ] AC2: 完了したタスクは未完了一覧から消える
- [ ] AC3: 完了操作は即座に反映される

**優先度:** Must

---

### US-004: タスクを削除する

**As a** 個人ユーザー
**I want** 不要なタスクを削除したい
**So that** 一覧をきれいに保てる

**Acceptance Criteria:**

- [ ] AC1: タスクの削除ボタンをクリックして削除できる
- [ ] AC2: 削除したタスクは一覧から消える

**優先度:** Must

---

### US-005: 完了済みタスクを確認する

**As a** 個人ユーザー
**I want** 完了したタスクの一覧を見たい
**So that** 何を終えたか振り返れる

**Acceptance Criteria:**

- [ ] AC1: 完了済みタスクの一覧を表示できる
- [ ] AC2: 完了済みタスクを未完了に戻せる

**優先度:** Should

---

### US-006: データが永続化される

**As a** 個人ユーザー
**I want** ブラウザを閉じてもタスクが残ってほしい
**So that** 毎回タスクを入力し直す必要がない

**Acceptance Criteria:**

- [ ] AC1: タスクの追加・完了・削除がブラウザのlocalStorageに保存される
- [ ] AC2: ページをリロードしてもタスクが復元される

**優先度:** Must
