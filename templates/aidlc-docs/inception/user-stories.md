# User Stories

## ACの品質基準

> すべての Acceptance Criteria は以下の3条件を満たすこと:
>
> 1. **具体的か**: 数値・条件・状態が明記されている
> 2. **検証可能か**: テストで合否を判定できる
> 3. **一意に解釈できるか**: 読む人によって解釈がブレない
>
> - NG: 「レスポンスが速いこと」→ OK: 「一覧画面の初回表示が2秒以内であること」
> - NG: 「適切なエラーメッセージを表示すること」→ OK: 「入力が空の場合、入力欄の下に『この項目は必須です』と赤字で表示すること」
